# 실행 계획 (Exec Plans)

특정 작업/이니셔티브를 진행하기 위한 단계별 계획과 진행 상황을 기록하는 디렉터리입니다. 전체 작업 목록과 우선순위는 [`PLANS.md`](../../PLANS.md)를 참고하세요.

## 구조

- `active/`: 진행 중인 계획
- `completed/`: 완료된 계획 (참고용 아카이브)

## 새 계획 작성 시 포함할 내용

[`completed/scaffolding.md`](completed/scaffolding.md)을 템플릿으로 참고하세요:

- 목표
- 진행 순서/단계 (해당하는 경우)
- 현재 상태 (체크리스트)
- 미해결 / 보류 항목
- 다음 작업

## 라이프사이클

1. 새 이니셔티브 시작 시 `active/`에 파일을 생성하고 [`PLANS.md`](../../PLANS.md)에 항목을 추가합니다.
2. 진행하며 "현재 상태"와 "다음 작업"을 갱신합니다.
3. 완료되면 `completed/`로 이동하고 `PLANS.md`의 상태를 갱신합니다.
