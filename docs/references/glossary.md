# 용어집 (Glossary)

`AGENTS.md`, `ARCHITECTURE.md` 등 다른 문서에서 사용하는 도메인 용어를 정리합니다. 새 용어가 등장하면 이 파일에 추가하세요.

| 용어 | 설명 |
|---|---|
| **WSI (Whole Slide Image)** | 디지털 스캐너로 유리 슬라이드 전체를 스캔한 초고해상도 이미지 (수백MB~수GB). |
| **DZI (Deep Zoom Image)** | WSI를 줌 레벨별 타일 피라미드로 분할한 형식. 뷰어가 필요한 영역/배율의 타일만 요청해 표시. |
| **MPP (Microns Per Pixel)** | 픽셀당 실제 크기(마이크론). 어노테이션 측정값(길이/면적)을 실제 단위로 환산하는 기준. |
| **배율 (Magnification)** | 스캔 시 대물렌즈 배율(예: 20x, 40x). 슬라이드의 해상도와 직결. |
| **Case (케이스)** | 하나의 진단 의뢰 단위 (접수번호 기준). 여러 Specimen을 포함할 수 있음. |
| **Specimen (검체)** | Case에 포함된 개별 조직 샘플 (예: 채취 부위 단위). |
| **Block (블록)** | Specimen으로부터 제작된 파라핀 블록. 하나의 Block에서 여러 Slide(염색별)가 만들어질 수 있음. |
| **Slide (슬라이드)** | 실제 스캔된 WSI 파일 + 메타데이터 (스캐너, 배율, MPP, 염색 종류 등). |
| **Annotation (어노테이션)** | 판독의가 슬라이드 위에 직접 그리는 영역/측정/메모. |
| **CellAnnotation (셀 어노테이션)** | AI 모델이 형태계측을 통해 생성하는 세포 단위 어노테이션 (슬라이드당 ~10만 건 규모). |
| **PHI/PII** | 환자를 식별할 수 있는 보호대상 정보 (이름, 생년월일, 환자번호 등). |
